## Appearance

### Default

Use `default` for the default dropdown menu appearance. Default menu has scroll after its height exceeds the pre-defined amount.

<Example
  Component={DropdownMenuStatelessDefault}
  packageName="@atlaskit/dropdown-menu"
/>

### Tall

Use `tall` to control the height of the menu. Tall menu has no scroll until the height exceeds the height of the viewport.

<Example
  Component={DropdownMenuStatelessTall}

/>

### Is menu fixed

Use `isMenuFixed` to determine if the menu should be in a fixed position. This is useful for breaking out of overflow scroll or hidden containers.
Extra layout management is needed to control scroll behavior when this property is enabled as the menu will not update the position with the target on scroll.
